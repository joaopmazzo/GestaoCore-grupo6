# GestaoCore - Grupo 6

**GestaoCore** é um sistema completo de gestão de clientes, produtos e estoque com arquitetura moderna baseada em React + Spring Boot.

## 📋 Resumo do Projeto

O projeto é composto por duas aplicações principais que trabalham em conjunto:

- **Frontend**: Interface web interativa desenvolvida com React + Vite, oferecendo um dashboard intuitivo com CRUD para clientes, produtos e usuários
- **API**: Backend REST com Spring Boot 4.0.5 e Java 21, implementando autenticação JWT, persistência com JPA/Hibernate e testes automatizados
